Asterisk 1.4.18-rc3 is now available. The important bug fixes that made it into this RC are a couple of crash fixes for ChanSpy/MixMonitor. A few other less severe bug fixes made it in, as well.

This release candidate is published for anyone that is interested in helping to test it for a couple of days before it is officially released. To download the release candidate, use the following svn command: `$ svn co http://svn.digium.com/svn/asterisk/tags/1.4.18 asterisk-1.4.18-rc3` If you would like it in tarball format, use the following commands: `$ svn export http://svn.digium.com/svn/asterisk/tags/1.4.18 asterisk-1.4.18-rc3 $ tar -czvf asterisk-1.4.18-rc3.tar.gz asterisk-1.4.18-rc3/` Thanks!
